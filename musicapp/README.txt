How to Run the Application
Follow these steps to set up and run the HarmonyStream application on your local machine.

Prerequisites
Before you begin, ensure you have the following installed:

Node.js (version 14 or higher recommended)

npm (Node Package Manager, usually comes with Node.js)

You can download Node.js from nodejs.org.

Installation
Clone the Repository (or create a new React project):
If you haven't already, create a new React project using Create React App:

npx create-react-app musicapp
cd musicapp

Copy the Code:

Open the src/App.js file in your harmony-stream project. Delete all its existing content.

Paste the entire content of the App.js code provided in the Canvas (Personalized Music Streaming UI/UX with Plain CSS (React) - Export Liked Songs) into your src/App.js file.

Open the src/App.css file in your harmony-stream project (create it if it doesn't exist). Delete all its existing content.

Paste the entire content of the App.css code provided in the Canvas (CSS for Personalized Music Streaming UI/UX (App.css)) into your src/App.css file.

Running the Application
Open your terminal or command prompt.

Navigate to your project directory:

cd musicapp

Start the development server:

npm start

This command will open the application in your default web browser at http://localhost:3000. The application will automatically reload if you make any changes to the source code.
